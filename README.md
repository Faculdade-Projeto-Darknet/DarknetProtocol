# SOBRE O PROJETO

O **Darknet Protocol** é um aplicativo Android com temática hacker/cyberpunk criado para fins educacionais.

O objetivo do projeto é ensinar conceitos básicos de:
- lógica computacional
- comandos de terminal
- análise de IP
- descriptografia simples
- resolução de desafios
- progressão de jogador
- persistência em nuvem

Tudo isso em um ambiente visual inspirado em interfaces cyberpunk e consoles futuristas.

Porque aprender segurança ofensiva em slide de PowerPoint deveria ser considerado crime digital.

COMO EXECUTAR

Clone o projeto:

git clone https://github.com/Faculdade-Projeto-Darknet/Darknet-Projeto.git

Abra no Android Studio e execute:

.\gradlew.bat assembleDebug

Ou rode diretamente pelo Android Studio.

Se o Gradle decidir sofrer emocionalmente, limpa o projeto e tenta de novo. Ritual clássico do Android.
